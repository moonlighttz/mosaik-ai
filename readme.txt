*About the application
Aplikasi yang kami kembangkan berupa sebuah aplikasi web-based yang bernama Mosaik.AI. Disini kami menerapkan konsep speech recognition sebagai implementasi dari proyek AI. Dengan aplikasi ini, kita dapat menerjemahkan bahasa Inggris ke bahasa Indonesia. Inputan yang dimasukkan berupa suara dan program akan secara otomatis mengenali apa yang diucapkan oleh user dan mengetiknya pada tampilan web. Setelah itu, kita dapat langsung menerjemahkan inputan yang sudah dimasukkan dan program akan menghasilkan output berupa terjemahan dari bahasa yang diinput tadi ke dalam bahasa Indonesia dan mengucapkan kata-katanya. 


*How to run :
1. Buka website Mosaik.AI (https://mosaik-ai.vercel.app/)
2. Pada tampilan web, click tombol Start Now yang akan diarahkan langsung ke section program penerjemah. Kita juga bisa scroll down untuk menuju section ini, section ini terdapat tepat dibawah section 'About'
3. Click tombol start untuk memulai perekaman. Allow access microphone untuk mengizinkan aplikasi merekam suara. Kata-kata yang diucapkan akan secara otomatis dideteksi dan muncul pada bagian textfield.
4. Click tombol Translate untuk menerjemahkan hasil dari inputan berupa voice tadi. (Text pada bagian textfield Speech)
5. Hasil translate akn otomatis ditampilkan pada bagian textfield Translation dan kata-kata pada textfield tersebut juga akan diucapkan satu per satu.
